# Polimi Networked Software for Distributed Systems (NSDS)
This repository contains exercises and resources related to the Networked Software for Distributed Systems (NSDS) course at Politecnico di Milano. The materials cover various topics in distributed systems, including:

- [Akka](docs/akka.md)
- [Apache Kafka](docs/kafka.md)
- [Node-RED](docs/node-red.md)
- [Apache Spark](docs/spark.md)
- [MPI](docs/mpi.md)
- [Contiki-ng](docs/contiki-ng.md)

## Repository Structure
- `docs/`: Documentation and theoretical materials for the course topics, they include my notes, integrated with code examples with the help of generative AI.
- `src/`: Source code for exercises and examples.
